HTML Tags
Reference

Basic Structure Tags

Tag Name Description
<html> HTML Tag The root element of an HTML document.
<head> Head Tag Contains meta-information about the document.
<title> Title Tag Sets the title of the document, shown in the browser’s title bar or tab.
<body> Body Tag Contains the content of the document.

Text Formatting Tags

Tag Name Description
<h1> to <h6> Heading Tags Define headings, with <h1> being the highest level and <h6> the lowest.
<p> Paragraph Tag Defines a paragraph.
<b> Bold Tag Makes text bold.
<i> Italicize Tag Makes text italic.
<u> Underline Tag Underlines text.
<a> Anchor Tag Defines a hyperlink.
<br> Line Break Tag Inserts a line break.
<hr> Horizontal Rule Tag Inserts a horizontal rule (line).

List Structure Tags

Tag Name Description
<ul> Unordered List Tag Defines an unordered list.
<ol> Ordered List Tag Defines an ordered list.
<li> List Tag Defines a list item.

Table Tags

Tag Name Description
<table> Table Tag Defines a table.
<tr> Table Row Tag Defines a table row.
<td> Table Cell Tag Defines a table cell.
<th> Table Header Tag Defines a table header cell.

Form Tags

Tag Name Description
<form> Form Tag Defines an HTML form for user input.
<input> Input Tag Defines an input control.
<label> Label Tag Defines a label for an <input> element.
<button> Button Tag Defines a clickable button.
<select> Select Tag Defines a drop-down list.
<option> Option Tag Defines an option in a drop-down list.

Media Tags

Tag Name Description
<img> Image Tag Embeds an image.
<audio> Audio Tag Embeds sound content.
<video> Video Tag Embeds video content.

Semantic Tags

Tag Name Description
<article> Article Tag Defines an article.
<section> Section Tag Defines a section in a document.
<nav> Nav Tag Defines navigation links.
<aside> Aside Tag Defines content aside from the main content.
<footer> Footer Tag Defines a footer for a document or section.
